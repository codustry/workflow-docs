## Overview

Warehouses represent physical storage locations. Inventory balances are tracked per product per warehouse.

<Accordion title="ภาษาไทย">
Warehouses หมายถึงสถานที่จัดเก็บจริง ยอดสินค้าคงคลังจะถูกติดตามแยกตามสินค้าและคลัง
</Accordion>

## Creating a Warehouse

1. Go to **Warehouses** → **New Warehouse**
2. Enter a name and optional description
3. Click **Save**

*ไปที่ **Warehouses** → **New Warehouse** กรอกชื่อและคำอธิบาย แล้วกด **Save***

## Inventory by Warehouse

From the **Inventory** module you can filter the stock balance by warehouse to see quantities per location.

*จากโมดูล **Inventory** คุณสามารถกรองยอดสต็อกตามคลังสินค้าได้*
