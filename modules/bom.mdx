## Overview

A Bill of Materials (BOM) defines the raw materials and components required to produce a finished product. BOMs are used when creating Work Orders.

<Accordion title="ภาษาไทย">
Bill of Materials (BOM) หรือสูตรการผลิต กำหนดวัตถุดิบและชิ้นส่วนที่ต้องใช้ในการผลิตสินค้าสำเร็จรูป BOM จะถูกใช้เมื่อสร้างใบสั่งผลิต (Work Order)
</Accordion>

## BOM Structure

| Field | Description |
|---|---|
| **Finished Product** | The product being produced |
| **Output Quantity** | How many units one BOM run produces |
| **Material Lines** | List of input materials with quantities |

## Creating a BOM

1. Go to **BOM** → **New BOM**
2. Select the finished product
3. Set the output quantity
4. Add material lines (product + quantity per line)
5. Click **Save**

*ไปที่ **BOM** → **New BOM** เลือกสินค้าสำเร็จรูป กำหนดจำนวนที่ผลิตได้ เพิ่มรายการวัตถุดิบแต่ละรายการ แล้วกด **Save***
