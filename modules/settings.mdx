## Overview

Organization Settings lets you configure your company profile and the automatic numbering sequences for all transaction documents.

<Accordion title="ภาษาไทย">
Organization Settings ช่วยให้คุณกำหนดโปรไฟล์บริษัทและรูปแบบเลขลำดับอัตโนมัติสำหรับเอกสารทุกประเภท
</Accordion>

## Company Profile

| Field | Description |
|---|---|
| **Company Name** | Displayed on documents |
| **Logo** | Company logo for PDFs |
| **Address** | Registered address |
| **Tax ID** | For invoicing |

## Number Sequences

Configure the prefix and starting number for each document type:

| Document | Example |
|---|---|
| Purchase Request | PR-2024-001 |
| Purchase Order | PO-2024-001 |
| Goods Receipt | GR-2024-001 |
| Work Order | WO-2024-001 |
| Delivery Note | DN-2024-001 |
| Stock Adjustment | SA-2024-001 |

*กำหนดรูปแบบและเลขเริ่มต้นสำหรับเอกสารแต่ละประเภท*
