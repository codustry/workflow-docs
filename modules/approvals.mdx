## Overview

The Approvals module shows all Purchase Requests pending your review. Managers and Admins can approve, reject, or return PRs.

<Accordion title="ภาษาไทย">
โมดูล Approvals แสดงใบขอซื้อทั้งหมดที่รอการตรวจสอบจากคุณ ผู้จัดการและ Admin สามารถอนุมัติ ปฏิเสธ หรือส่งคืนได้
</Accordion>

## Actions

| Action | Description |
|---|---|
| **Approve** | PR is approved and ready to convert to PO |
| **Reject** | PR is denied — requester is notified |
| **Return** | PR is sent back to requester for revision |

## Approving a PR

1. Go to **Approvals**
2. Click on a PR to review its details and line items
3. Select **Approve**, **Reject**, or **Return**
4. Add a comment if needed
5. Confirm

*ไปที่ **Approvals** คลิกที่ PR เพื่อดูรายละเอียด จากนั้นเลือก **Approve**, **Reject** หรือ **Return*** 

## Who Can Approve?

Users with the **Admin** or **Manager** role can approve PRs.

*ผู้ใช้ที่มีบทบาท **Admin** หรือ **Manager** สามารถอนุมัติ PR ได้*
