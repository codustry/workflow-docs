## Overview

The Users module lets administrators invite team members and assign roles. See [Roles & Permissions](/development) for the full permissions matrix.

<Accordion title="ภาษาไทย">
โมดูล Users ช่วยให้ผู้ดูแลระบบเชิญสมาชิกทีมและกำหนดบทบาท ดู [Roles & Permissions](/development) สำหรับตารางสิทธิ์ทั้งหมด
</Accordion>

## Inviting a User

1. Go to **Settings** → **Users** → **Invite User**
2. Enter the user's email address
3. Select one or more roles
4. Click **Send Invite**

The user will receive an email with a link to set their password.

*ไปที่ **Settings** → **Users** → **Invite User** กรอกอีเมล เลือกบทบาท แล้วกด **Send Invite** ผู้ใช้จะได้รับอีเมลพร้อมลิงก์ตั้งรหัสผ่าน*

## Editing a User

Click on any user to edit their roles or deactivate their account.

*คลิกที่ผู้ใช้เพื่อแก้ไขบทบาทหรือปิดใช้งานบัญชี*
