## Overview

The Vendors module stores all supplier information used when creating Purchase Requests and Purchase Orders.

<Accordion title="ภาษาไทย">
โมดูล Vendors เก็บข้อมูลผู้จำหน่ายทั้งหมดที่ใช้ในการสร้างใบขอซื้อและใบสั่งซื้อ
</Accordion>

## Key Fields

| Field | Description |
|---|---|
| **Name** | Supplier / company name |
| **Country** | Supplier country |
| **Currency** | Default currency for POs |
| **Payment Terms** | e.g., Net 30, Net 60 |
| **Contact** | Contact person and details |

## Creating a Vendor

1. Go to **Vendors** → **New Vendor**
2. Enter supplier name, country, currency, and payment terms
3. Click **Save**

*ไปที่ **Vendors** → **New Vendor** กรอกชื่อ ประเทศ สกุลเงิน และเงื่อนไขชำระเงิน แล้วกด **Save***
