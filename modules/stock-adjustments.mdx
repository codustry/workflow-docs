## Overview

Stock Adjustments allow authorized users to correct inventory levels through counted, increase, decrease, or scrap adjustments.

<Accordion title="ภาษาไทย">
Stock Adjustments อนุญาตให้ผู้ใช้ที่มีสิทธิ์ปรับแก้ระดับสินค้าคงคลังด้วยการนับ เพิ่ม ลด หรือตัดทิ้ง
</Accordion>

## Adjustment Types

| Type | Description |
|---|---|
| **Count** | Set stock to a specific quantity (physical count result) |
| **Increase** | Add quantity to existing stock |
| **Decrease** | Remove quantity from existing stock |
| **Scrap** | Write off damaged or unusable stock |

## Creating an Adjustment

1. Go to **Inventory** → **Stock Adjustments** → **New Adjustment**
2. Select warehouse and product
3. Select adjustment type
4. Enter quantity and reason
5. Click **Post**

*ไปที่ **Inventory** → **Stock Adjustments** → **New Adjustment** เลือกคลัง สินค้า ประเภทการปรับ กรอกจำนวนและเหตุผล แล้วกด **Post***

<Warning>
Posted adjustments are permanent and create an audit trail entry.

*การปรับที่ Post แล้วจะถาวรและสร้างบันทึกการเปลี่ยนแปลง*
</Warning>
